ade25.banner Installation
-------------------------

To install ade25.banner using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ade25.banner`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ade25.banner
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
